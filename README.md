# Ahirutochi 🦆💥

A game about ducks, brewing potions and explosions.

Made in 48h for the [Global Game Jam 2025](https://globalgamejam.org/).
Play on [eddex.itch.io/ahirutochi](https://eddex.itch.io/ahirutochi)!
